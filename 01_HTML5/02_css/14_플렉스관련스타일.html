        선택자{

        * 요소를 통해 레이아웃을 구성시 별다른 사이즈, 크기, 정렬을 지정하지 않아도 유연하게(flex) 처리하는 속성

justify-content : 요소들을 "가로" 선상에서 정렬하도록 도와주는 속성
align-content : 요소들을 "세로" 선상에서 정렬하도록 도와주는 속성(단, justify-content와 다르게 한 줄 때면 적용X) 

        선택자 {
            justify-content : flex-start | flex-end | center | space-between | space-evenly | space-around
        }
        
        선택자 {
            align-content : flex-start | flex-end | center | space-between | space-evenly | space-around
        }

flex-direction : flex요소들의 정렬 방향을 지정(행, 열)

        선택자 {
            flex-direction : row | row-reverse | column | column-reverse;
        }

        + 만약에 flex-direction의 값이 column 혹은 column-reverse가 되면 justify-content의 기준이 세로, align-items의 기준이 가로 바뀜

align-items : 요소들을 "세로" 선상에서 정렬하는 속성
+ align-self : align-items과 명령어는 똑같이 쓰이지만 각각의 요소들을 따로 지정 가능 

        선택자 {
            align-items : flex-start | flex-end | center | baseline | stretch
