Content 영역 관련 메소드

* html(["문구"]) : innerHTML 속성과 관련된 메소드

선택된 요소의 content 영역의 값을 리턴해주거나 또는 변경해주는 메소드
-> $("선택자").html() : 선택된 요소의 content 영역의 값을 반환해줌 (html 태그까지 포함)
-> $("선택자").html("문구") : 선택된 요소의 content영역의 값을 해당 문구로 변경 (html 태그가 존재 할 경우 태그로써 해석해줌)

네이버로

* text(["문구"]) : innerText 속성과 관련된 메소드

선택된 요소의 content 영역의 값을 리턴해주거나 변경해주는 메소드
-> $("선택자").text() : 선택된 요소의 content 영역을 반환(단, html 태그 포함하지 않고, 오로지 text만) -> $("선택자").text("문구") : 선택된 요소의 content 영역을 전달값으로 변경(단, html 태그가 존재할 경우 태그로 해석 X)
