jQuery 개요

* jQuery란?

기존에 자바스크립트 언어만으로 구현하기에는 복잡해짐
그래서 구문을 간소화 하기 위해 "존 레식"에 의해 만들어짐
Javascript 기반 라이브러리(유용한 기능들의 모음집)
즉, Javascript 언어로 유용한 함수나 내용들이 이미 정의돼있는 모음집

복잡한 자바스크립트 구문을 보다 짧고 쉽게 개발 가능
적은 양의 소스코드로 빠르고, 풍부한 기능을 제공

>> jQuery 장점
1) DOM 요소 => HTML태그들과 관련된 스크립트를 보다 쉽게 구현 가능
2) AJAX(비동기 통신), 이벤트 처리 등등 폭넓게 지원
3) jQuery와 관련된 확장형 플러그인, 오픈 api 등을 지원

* jQuery 라이브러리 연결 방법

head 태그 내에 script 태그로 외부 jQuery관련한 .js 파일 연결
오프라인(비네트워크) / 온라인(네트워크) 방식
- jQuery 공식 사이트 : jQuery 공식 사이트
1. jQuery 라이브러리 직접 다운로드 받은 후 경로 지정 (오프라인)
2. CDN(content Delievery Network) 방식(온라인)
파일을 다운로드 받아서 저장할 필요 없이 라이브러리를 제공하는 사이트 url 제시해서 파일 경로 지정

* jQuery 구문 작성하기

jQuery 구문으로 DOM 요소를 다루는 구문은
문서상에 요소들이 다 만들어지고 나서 실행해야함

>순수 자바스크립트 

                window.onload = function(){
                    실행내용;
                }

                Javascript에서의 window.onload 구문은 해당 문서의 요소들이 다 로드되고 
                + 외부문서들(css파일)도 다 가지고 온 뒤에 실행됨
                그리고 window.onload 구문은 해당 이 문서에서 단 한번만 작성 가능
>jQuery에서의 내용은 3가지 구문 제공 
                1)jQuery(document).ready(function(){
                    실행 내용;
                })

                2)$(document).ready(function(){
                    실행 내용;
                })

                3)$(function(){
                    실행 내용;
                })

                해당 문서의 요소들만 로드 되면 곧바로 실행 가능
                여러번 기술 가능(제한없음) => 순서대로 실행
