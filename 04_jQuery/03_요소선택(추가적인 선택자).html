* 속성 선택자

선택자[속성] : 특정 속성을 가지고 잇는 모든 요소 선택
선택자[속성 = 특정값] : 속성값이 특정값과 "일치하는 모든" 선택자[속성 ~= 특정값] : 송성값에 특정값을 "단어로써 포함"하는 선택자[속성 ^= 특정값] : 송석값이 특정값으로 "시작하는 요소" 선택자[속성 $= 특정값] : 속성값이 특정값으로 "끝"나는 요소 선택
선택자[속성 *= 특정값] : 속성값이 특정값을 "포함"하는 요소
