배열

변수 선언시, 별도의 자료형 지정이 없으므로
어떤 자료형의 값들이든 하나의 공간에 담을 수 있고, 크기 제약도 없음.

배열의 선언

배열 관련 메소드

1_ indexOF(찾고자 하는 요소) : 배열에서 해당 요소가 위치해있는 인덱스 반환

배열.concat(배열1, 배열2, ...) : 배열에다가 여러 개의 배열을 결합하고자 할 때 사용

3) 배열.reverse() : 배열에 담긴 값을 역순으로 바꿔주는 메소드

4) 배열.sort() : 배열안에 담긴 값들 오름차순 정렬시켜주는 메소드

5_1) 배열.push(추가할요소 ) : 배열의 맨 뒤에 요소를 추가키시고 배열의 크기반환
5_2) 배열.pop() : 배열의 맨 뒤 요소 제거시키고 제거된 요소 반환

6_1) 배열.unshift(추가할요소) : 배열의 맨 앞에 요소 추가 후 배열의 크기 반환
6_2) 배열.shift() : 배열의 맨 앞에 요소 제거 시키고 제거된 요소 반환

7_1) 배열.slice(시작인덱스, 끝인덱스) : 배열로부터 시작인덱스에서 부터 끝 인덱스 까지 단지 추출해주는 메소드
7_2) 배열.splice(시작인덱스, 제거개수, 추가할값(생략가능)) : 배열의 요소 추출해서 제거 및 추가

8) 배열.toString() / 배열.join([구분자]) : 배열에 각 인덱스 담긴 값들을 하나의 "문자열"로 합쳐서 반환
