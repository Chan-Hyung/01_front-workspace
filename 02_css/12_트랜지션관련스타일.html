트랜지션 적용

태그가 변형속성에 의해 변형이 될 때 부드럽게 변형이 될 수 있게 해주는 속성 transition-property : 전환효과를 사용할 속성 이름 (all, 전환효과를 사용할 속성 이름)
transition-duration : 전환효과의 지속시간
transition-timing-function : ease(느리게 -> 빠르게 -> 느리게), linear(일정하게)
transition : 모든 속성을 한 번에 지정할 때 사용 (속성명 지속시간 타이밍함수)
