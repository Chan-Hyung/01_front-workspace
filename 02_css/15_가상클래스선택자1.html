가상 클래스 선택자1

- 어떠한 행동을 했을 때 동작하는 개념을 선택하는 선택자
- 종류 : hover, active, focus



hover 가상 선택자

- 마우스 커서가 올라가 있는 동안 선택

        선택자:hover {
            스타일속성: 값;
        }

active 가상 선택자

- 마우스를 클릭하고 있는 동안 선택

        선택자:active{
            스타일속성:값;
        }

focus 가상 선택자

- 요소가 포커스 되면 선택
- 포커스 가능한 요소 : input, a, button, label 등...



tabindex

- Tab 키를 사용해서 요소를 선택 할 수 있느 순서 지정 가능 - focus가 아닌 요소를 선택 가능한 요소로 만드는 속성 

        tabindex= "-1" (-1 이외에 다른 값을 넣는 것은 논리적 흐름을 방해하기 때문에 권장하지 않음)
